# 梦幻优化服务端

本整合包专为 Minecraft 服务器设计，提供基于 Paper 和 Purpur 核心的全面性能优化配置，覆盖主流版本（1.21.4至1.19.4），旨在显著提升服务器运行效率与稳定性。



增强区块加载：更快更高效的区块加载，带来无延迟体验。



优化实体：减少实体延迟，让游戏更流畅。



优化红石：更高效的红石机制，最大限度减少延迟。
